# Gravel PRO Library

![Status](https://img.shields.io/badge/status-active-success)
![License](https://img.shields.io/badge/license-MIT-blue)
![Platform](https://img.shields.io/badge/platform-Zwift%20%7C%20Garmin%20%7C%20Intervals.icu-orange)

Professional open-source workout library for gravel and endurance cycling.

## Vision

Create one of the most comprehensive collections of structured cycling workouts for:

- Gravel Racing
- Ultra Gravel
- Marathon MTB
- Road Endurance

Compatible with:

- Zwift (.zwo)
- Garmin Edge
- Intervals.icu
- TrainingPeaks
